# LEAF-Toolbox-Canada

Google Earth Engine implementation of algorithms for producing Level 2b Vegetation Biophysical Parameter maps for Canada.

Current Biophsyical Parameters are:

Fraction of Absorbed Photosynthetically Active Radiation
Fraction of Canopy Cover
Leaf Area Index

Current algorithms are:
SL2P (Weiss and Baret, 2016)

Refer to WIKI for use instructions and Product User Guides.
Refer to github.com/rfernand387/LEAF-Toolbox for an equivalent production system for user specified temooral and spatial extents.
